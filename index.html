Guess The Number

This was my first Javascript Challenge. Coding this game really grounded my understanding of javascript. The game randomly generates a number from 1-100 and prompts the user to guess what it is. It then prompts the user wether their guess is too high, too low, or correct and stores their guess history. The user can reset the game at any time which causes the game to generate a new random number and clear the guess history. You can view the code I used for this project
